# Kapis Clinic AI - Admin Console

Angular 20 admin console for Kapis Clinic AI. Part of the [kapis-ai-platform](../../README.md)
monorepo - see the root README for the full local setup (Docker stack, env vars) and
`../../docs/` for architecture, folder structure, dev workflow, and coding standards.

## Quick reference

```bash
npm install
npm start              # http://localhost:4200
npm run build           # production build -> dist/clinic-admin
npm test                # Karma/Jasmine unit tests
npm run lint            # ESLint
npm run format           # Prettier (writes changes)
```

Generated with [Angular CLI](https://angular.dev/tools/cli) 20.
